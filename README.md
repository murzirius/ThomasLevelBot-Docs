# ThomasLevelBot-Docs

## 🧭 Навигация - https://github.com/murzirius/ThomasLevelBot-Docs/wiki

## 🐞 Сообщить об ошибке - https://github.com/murzirius/ThomasLevelBot-Docs/issues

Бот помогает отслеживать и мотивировать некоторых пользователей активно общаться в чате.

Пользователи получают опыт (XP) за сообщения. Максимальное количество XP за сообщение - 25 (или 50 с премиумом). Количество получаемого опыта зависит от длины сообщения. В целях защиты от накрутки опыта и спама, опыт может быть начислен 1 раз в 30 секунд.

❓️ В этом репозитории не будет опубликовано никаких скриптов.
